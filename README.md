# Features

List of all the features that can be found on the current version of the plugin:

## Spritesheet Editor

- Padder
  - Padding: Add padding around all tiles on a spritesheet.
  - Anti-bleed pixels: Fills the padding with a copy of the border-touching pixels to fight pixel bleed.
- Animation Expoter
  - Export animations as spritesheets.
